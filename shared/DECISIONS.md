# Shared Decisions

## Purpose

This file records **cross-product decisions only**. Product-specific decisions live in each product’s DECISIONS file.

## Decision template

For each entry use:

| Field | Description |
|-------|-------------|
| **Date** | When the decision was made |
| **Decision** | What was decided |
| **Context** | Why it was needed |
| **Aligned With** | Who agreed or was consulted |
| **Impact** | What is affected (products, teams, systems) |
