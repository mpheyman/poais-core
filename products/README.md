# Products

## How to create a product

1. Create a folder: `products/<product-name>/`
2. Use lowercase, hyphenated names (e.g. `products/my-product/`).
3. Add the minimum required files (see below).

## Minimum required product files

Each product folder must contain:

- **CONTEXT** — product context and background
- **PLAN** — plan and approach
- **EXECUTION** — execution tracking and delivery
- **DECISIONS** — decisions and rationale
- **STATUS** — current status (updated weekly)

## Recommended additional files

- **DISCOVERY** — discovery and research
- **RISKS** — risks and mitigation
- **ROADMAP** — roadmap and milestones
- **FEATURES/** — folder for feature-level docs
- **MEETINGS/** — folder for meeting notes
- **INPUTS/** — folder for external inputs and references
