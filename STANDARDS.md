# Standards

- **Every product** must have: CONTEXT, PLAN, EXECUTION, DECISIONS, STATUS. Other artifacts (DISCOVERY, RISKS, ROADMAP, FEATURES/, MEETINGS/, INPUTS/, IDEAS/) are encouraged and flexible. ROADMAP has a required Milestones section and is fed by PLAN, EXECUTION, DECISIONS. **Portfolio repos** have multiple product dirs (e.g. `products/<name>/`) and optionally **portfolio/** with PRIORITIES.md and STATUS.md (roll-up from `/status portfolio`).
- **Decisions** are logged with: Date, Decision, Context, Aligned With, Impact.
- **STATUS** is updated weekly (even if brief).
- **POAIS reflects reality**: if execution diverges from PLAN, update PLAN; if scope changes, log a decision; if risk emerges, add to RISKS.
